# if 
if : It is use for check the only one condition.  
**syntax** 
```  
if condition:   
    statement
```    
# if..else 
if..else : It is use for check one condition otherwise control pass to else statement.  
**syntax** 
```  
if condition:   
        statement  
else:   
       statement
```
# if..elif  
if..elif : It is use for check more then one condition.   

**syntax** 
```
if condition:   
        statement  
elif:   
        statement
```
